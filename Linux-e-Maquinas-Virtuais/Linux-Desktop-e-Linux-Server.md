<H1><b> Diferenças entre Linux Desktop e Linux Servers <b></H1>
<p >
  Temos aqui muitas diferenças entre os 2, porém vamos falar de coisas mais impactantes no ambiente de trabalho e ou dia-a-dia.<br>
  O Sistema operacional que tem como base ser utilizado no seu cotidiano é o famoso "desktop", le é uma versão mais completa do S.O,
  ele vai ter ambiente grafico que você pode usar o mouse (principal e impactante diferença) o que é muito comum em ambientes que o usuário
  usa para navegar na internet e manipular arquivos, claro que algumas ferramentas já vem instaladas para essa comodidade, como por exemplo o libreoffice (software opensource) que faz a edição de arquivos igual ao famoso "Pacote Office" da famosa janela (windows).<br>
  O Ambiente para servidores (Servers) é uma interface que é bem mais complexa para administrar caso não tenha tido contato com o sistema, nele vem instalado o menos de ferramentas possíveis, além de você usar diretamente a linha de comando (CLI) para poder administrar arquivos e interagir com ferramentas e devido a isso não mouse para interação do usuário, isso se deve por não ter o ambiente grafico nele instalado (o que podemos facilmente instalar caso precisemos), então a versão "servers" é para um usuário mais intermédiario/avançado.
</p>
